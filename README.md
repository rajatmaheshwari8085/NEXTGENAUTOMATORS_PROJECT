## 🛠️ Project Components
- **Soil Moisture Sensor** – Detects soil conditions and automates irrigation systems.
- **Ultrasonic Sensor (HC-SR04)** – Identifies movement and enhances field security.
- **DHT Sensor** – Monitors temperature and humidity, enabling automatic control of devices like cooling fans.
- **Web-based Dashboard** – Displays live data and allows users to control appliances remotely.

## 💻 Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, Javascript, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)

## ⚙️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/rajatmaheshwari8085/NEXTGENAUTOMATORS_PROJECT.git
   cd NEXTGENAUTOMATORS_PROJECT
   ```

2. Start the Frontend Server:
   ```bash
   cd NextGenAutomators
   npm install
   node server.js
   ```

3. Open your browser and visit `http://localhost:3000`.
